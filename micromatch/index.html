MicroMatch

Ham Radio Contact Coordination App

MicroMatch is a React Native Web and Mobile application designed to help amateur radio operators plan and coordinate microwave contacts. Built with modern web technologies while maintaining a focus on functionality and ease of use.

Features

  • Contact coordination and scheduling
  • Radio operator management
  • Cross-platform compatibility (web, mobile)
  • Real-time updates and notifications
  • Simple, intuitive interface

Technical Details

Built using React Native Web with Expo, MicroMatch provides a seamless experience across different platforms. The app uses modern web standards while maintaining compatibility with older browsers.
